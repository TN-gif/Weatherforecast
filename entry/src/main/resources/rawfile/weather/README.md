# 天气视频资源说明

## 视频文件规范
- **分辨率**: 1080p (1920x1080)
- **格式**: H.264编码MP4文件
- **码率**: ≤5Mbps
- **时长**: 10-15秒无缝循环
- **文件大小**: ≤5MB

## 当前需要的视频文件
- `sunny_day.mp4` - 晴天日景视频
- `sunny_night.mp4` - 晴天夜景视频
- `rainy_day.mp4` - 雨天日景视频
- `rainy_night.mp4` - 雨天夜景视频
- `snow_day.mp4` - 雪天日景视频
- `snow_night.mp4` - 雪天夜景视频
- `night.mp4` - 通用夜景视频

## WebP动画规范
- **尺寸**: ≤屏幕宽高80%
- **帧率**: 30fps
- **循环**: 无限循环
- **透明度**: 保留30%透明度通道

## 当前需要的WebP文件
- `sunny.webp` - 晴天动画
- `rainy.webp` - 雨天动画
- `snow.webp` - 雪天动画
- `night.webp` - 夜景动画

## 获取资源建议
1. 使用免费视频素材网站（如Pixabay、Pexels）
2. 使用AI生成工具创建循环视频
3. 使用After Effects等工具制作WebP动画
4. 确保所有素材符合商用授权要求
